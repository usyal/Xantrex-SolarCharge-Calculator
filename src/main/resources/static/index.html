How The Application Works

Required Inputs

  • Maximum Power of a single panel (Pmax) in Watts
  • Open Circuit Voltage of a single panel (Voc) in Volts
  • Short Circuit Charge of a single panel (Isc) in Amps
  • Number of Panels in Series Ns and Parallel NP
  • Battery Bank Voltage Vnom of Solar Array

Computational Formulas

Total Array Power = Pmax * Ns * Np

Open Circuit Voltage = Voc * Ns * temp. correction factor

Max Current Charge = Total Array Power / Vnom

*Default temp. correction factor is set to 1.2
